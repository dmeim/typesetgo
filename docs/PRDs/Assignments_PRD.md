# Feature: Assignments

## 1. Overview
Assignments allow teachers to prescribe specific work to students with due dates and completion criteria.

## 2. Assignment Lifecycle
* **Creation**:
    * Select a **Preset** (the task).
    * Select **Target**: Class or specific students.
    * Set **Due Date/Time**.
    * **Scoring Rule**: `Best Attempt`, `First Attempt`, or `Average`.
    * **Strict Mode**: Optional. Hides detailed results until teacher releases them.
* **Student View**:
    * Dashboard tile showing "To Do" assignments.
    * Status indicators: `Not Started`, `In Progress`, `Completed`, `Overdue`.
    * "Start" button launches the locked preset.
* **Completion**:
    * Automatically marked complete when criteria met.
    * **Auto-makeup**: Optional rule to allow late submissions (flagged).

## 3. Notifications
* Email and in-app reminders:
    * New Assignment.
    * Due in 24h.
    * Overdue.

## 4. Grading & Analytics
* **Teacher Gradebook**:
    * Matrix view: Students vs Assignments.
    * Cells show: Status, Score (WPM/Acc), Submission Time.
* **Drill-down**: View specific attempt associated with the grade.
